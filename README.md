# Swiggy Clone 🍔

A fully responsive Swiggy landing page clone built using **HTML, CSS, and JavaScript**.

## 📌 Features

* Responsive design
* Modern navigation bar
* Hero section
* Food category section
* Restaurant cards
* Footer
* Clean and organized code
* Mobile-friendly layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Swiggy-clone-web/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Swiggy-clone-web.git
```

2. Open the project folder.

3. Open `index.html` in your browser or use VS Code Live Server.
